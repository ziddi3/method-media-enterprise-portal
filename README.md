# Method Media Enterprise Agreement Portal

A professional digital agreement portal with cyberpunk/fantasy aesthetics, featuring dynamic pricing configuration, signature capture, and PDF agreement generation.

## Features

- 🎨 **Artistic UI** - Cyberpunk/fantasy aesthetic with neon particles, energy ribbons, and mandala glows
- ✍️ **Signature Capture** - Canvas-based signature pad that creates unique wax seals
- 💰 **Dynamic Pricing** - Interactive service configurator with finalized totals
- 📄 **PDF Generation** - Professional PDF agreement documents
- 🔐 **Authentication** - Session-based authentication with signature passkeys
- 👨‍💼 **Admin Dashboard** - Admin panel for managing agreements

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Node.js, Express.js
- **Database**: SQLite
- **PDF**: PDFKit

## Installation

```bash
npm install
npm start
```

## Environment Variables

Create a `.env` file with:
```
PORT=3000
SESSION_SECRET=your-secret-key
```
